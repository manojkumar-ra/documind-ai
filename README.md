# DocuMind AI

A production RAG application that allows users to 
upload documents and ask questions using AI.

## Features
- Upload any text document
- Ask questions about your document
- AI answers based on your document content

## Tech Stack
- Python
- Streamlit
- Groq LLM (llama-3.3-70b-versatile)
- HuggingFace Embeddings
- ChromaDB Vector Database
- LangChain

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Add your Groq API key in app.py
4. Run: streamlit run app.py

## Setup
Get your free Groq API key at console.groq.com
